# 🐶 DOGDOG 프로젝트 개요

## 🎯 프로젝트 목적
반려견 사료 관리 및 상태 모니터링을 위한 통합 서비스입니다.
고객(집사)이 반려견의 급여 기록과 사료 재고 데이터 등을 체계적으로 관리할 수 있도록 돕는 것을 목표로 합니다.

## 🏗️ 주요 기능
- **반려견 급여 관리**: 급여 기록(사료 종류, 양, 시간, 칼로리)을 등록하고 조회, 수정, 삭제합니다.
- **사료 재고 자동 업데이트**: 급여 기록에 맞춰 잔여 사료량을 계산하고 예측합니다.
- **칼로리 기반 영양 분석**: 각 반려견의 기초 대사량과 섭취 칼로리를 비교합니다.

## 👥 사용자 역할
- **일반 보호자 (Customer)**: 자신의 반려견과 사료를 관리합니다.
- **관리자 (ERP)**: 서비스 전체 데이터와 통계를 관리합니다.
